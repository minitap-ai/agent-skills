# Minitest CLI

The canonical skill for the minitest CLI lives in the
[minitest-cli repository](https://github.com/minitap-ai/minitest-cli/blob/main/SKILL.md)
so it stays in sync with the CLI itself.

To fetch the latest version, run:

```bash
minitest skill
```

This prints the up-to-date skill content to stdout.
